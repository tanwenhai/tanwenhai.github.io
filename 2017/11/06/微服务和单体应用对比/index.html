微服务和单体应用对比

单体应用容易部署、测试随功能演进越来越臃肿，可维护性、灵活性逐渐降低，维护成本越来越高

  1. 复杂性高
  2. 技术债务
  3. 可靠性差
  4. 扩展能力受限
  5. 阻碍技术创新

微服务具有以下优点

  1. 易于开发和维护
  2. 单个微服务启动较快
  3. 局部修改容易部署
  4. 技术栈不受限
  5. 按需伸缩
